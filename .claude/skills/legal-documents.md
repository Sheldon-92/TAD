## 合同审阅框架

### 审阅清单

```markdown
## 合同审阅清单

### 1. 基本信息
- [ ] 合同各方名称和法律地位
- [ ] 签署日期和生效日期
- [ ] 合同期限（固定/自动续约）
- [ ] 管辖法律和争议解决

### 2. 商业条款
- [ ] 服务/产品范围明确
- [ ] 价格和付款条件
- [ ] 交付时间和验收标准
- [ ] 违约责任和赔偿

### 3. 风险条款
- [ ] 责任限制条款
- [ ] 赔偿条款 (Indemnification)
- [ ] 保密义务
- [ ] 知识产权归属

### 4. 终止条款
- [ ] 终止条件
- [ ] 提前通知期限
- [ ] 终止后义务
- [ ] 退款/结算条款

### 5. 特殊条款
- [ ] 排他性条款
- [ ] 竞业禁止
- [ ] 不可抗力
- [ ] 转让限制
```

## 常用合同模板

### NDA（保密协议）要点

```markdown
## 保密协议 (NDA) 核心要素

### 1. 定义条款
- 保密信息的范围和定义
- 排除事项（公开信息、独立开发等）

### 2. 保密义务
- 保密期限（通常 2-5 年）
- 使用限制（仅限特定目的）
- 信息保护措施要求

### 3. 披露例外
- 法律强制披露
- 向顾问披露（需签署同等 NDA）

### 4. 返还/销毁
- 协议终止后的信息处理
- 返还或销毁的时限

### 5. 违约救济
- 禁令救济的可行性
- 损害赔偿
```

### 服务协议要点

```markdown
## 服务协议核心要素

### 1. 服务范围
- 具体服务内容描述
- 交付成果定义
- 服务级别协议 (SLA)

### 2. 费用与付款
- 定价模式（固定/时间/里程碑）
- 付款时间和方式
- 逾期付款的后果

### 3. 知识产权
- 背景 IP 归属
- 新创造 IP 归属
- 许可范围

### 4. 保证与责任
- 服务质量保证
- 责任限额
- 免责事项

### 5. 变更管理
- 变更请求流程
- 费用调整机制
```

## 条款库

### 常用法律条款

```markdown
## 标准条款库

### 不可抗力条款
> 因战争、自然灾害、政府行为、流行病或其他不可预见、
> 不可避免且不可克服的客观情况，导致本协议无法履行的，
> 受影响方不承担违约责任。受影响方应在事件发生后[X]天内
> 通知对方，并提供相关证明材料。

### 争议解决条款（仲裁）
> 因本协议引起的或与本协议有关的任何争议，应提交至
> [仲裁机构名称]，按照该机构届时有效的仲裁规则进行仲裁。
> 仲裁地点为[城市]，仲裁语言为[语言]。仲裁裁决是终局的，
> 对双方均具有约束力。

### 责任限制条款
> 除故意或重大过失外，任何一方因本协议产生的责任
> 不超过该方在违约事件发生前[12]个月内根据本协议
> 实际支付或应收的费用总额。任何一方均不对间接损失、
> 利润损失或数据丢失承担责任。

### 保密条款
> 各方对在本协议履行过程中获取的对方商业秘密和
> 其他保密信息负有保密义务。未经信息提供方书面同意，
> 接收方不得向第三方披露或用于本协议目的之外的用途。
> 本保密义务在协议终止后[X]年内继续有效。
```

## 最佳实践

```
✅ 推荐
□ 始终标注需要律师确认的事项
□ 保留修改痕迹（红线版本）
□ 记录所有关键日期
□ 使用标准化的条款库
□ 进行版本控制

❌ 避免
□ 替代专业法律意见
□ 忽视管辖法律差异
□ 遗漏隐含的义务条款
□ 忽视关联协议的影响
```

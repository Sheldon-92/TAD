## TL;DR Quick Checklist

```
1. [ ] 明确人群/场景/目标；用 AIDA/PAS/FAB 结构化表达
2. [ ] 标题10法：数字/问题/指南/对比/秘密/错误/清单/教程/故事/口号
3. [ ] 核心价值→优势→利益→CTA（含承诺与门槛）
4. [ ] SEO 关键词自然分布；避免关键词堆砌；元信息完善
5. [ ] 变体与测试计划（A/B），记录指标与结论
```

**Red Flags:**
- 空泛口号；堆砌功能；无场景/无用户语言；无CTA

## 文案公式

### AIDA 模型

```markdown
## AIDA 文案结构

**Attention (注意)** - 抓住眼球
- 震撼的数据
- 反常识的观点
- 强烈的视觉

**Interest (兴趣)** - 引发好奇
- 痛点共鸣
- 故事引入
- 利益承诺

**Desire (欲望)** - 激发渴望
- 场景描绘
- 社会证明
- 情感触动

**Action (行动)** - 促成转化
- 明确的 CTA
- 紧迫感
- 降低门槛

---
示例:

【注意】90% 的创业者都在犯这个致命错误
【兴趣】你的营销预算是不是总觉得花不到点上？
【欲望】我们帮助 500+ 企业实现了 3 倍 ROI 增长
【行动】免费领取《营销诊断报告》→ 立即申请
```

### PAS 模型

```markdown
## PAS 文案结构

**Problem (问题)** - 指出痛点
**Agitation (激化)** - 放大问题
**Solution (解决)** - 给出方案

---
示例:

【问题】你是否每天加班到深夜，却看不到项目进展？

【激化】低效的协作正在消耗你的时间、精力和热情。
每次开会都在重复沟通，每次交付都在返工修改。
长此以往，不仅项目延期，团队士气也在下降。

【解决】ProjectX 帮你告别混乱。智能任务分配、
实时进度同步、自动生成报告。已有 10000+ 团队
节省了 40% 的协作时间。免费试用 14 天 →
```

### FAB 模型

```markdown
## FAB 文案结构

**Feature (特性)** - 产品有什么
**Advantage (优势)** - 比别人好在哪
**Benefit (利益)** - 对用户意味着什么

---
示例:

| 特性 | 优势 | 利益 |
|------|------|------|
| AI 智能推荐 | 精准度 95% | 每月节省 10 小时选品时间 |
| 一键同步库存 | 支持 50+ 平台 | 告别超卖和缺货烦恼 |
| 可视化数据看板 | 实时更新 | 随时掌握经营全貌 |
```

## 标题写作

### 高转化标题公式

```markdown
## 10 个高转化标题公式

1. **数字型**
   "7 个方法让你的转化率翻倍"
   "[数字] + [方法] + [结果]"

2. **问题型**
   "为什么你的广告投放总是亏钱？"
   "为什么 + [痛点]？"

3. **指南型**
   "2024 年小红书运营完全指南"
   "[时间] + [主题] + 完全指南"

4. **对比型**
   "新手 vs 高手：写文案的 5 个差距"
   "[A] vs [B]：[差异点]"

5. **秘密型**
   "顶级投手不会告诉你的 3 个秘密"
   "[权威] + 不会告诉你 + [内容]"

6. **错误型**
   "90% 的人都在犯的投放错误"
   "[比例] + 都在犯的 + [错误]"

7. **清单型**
   "必备清单：开店前需要准备的 20 件事"
   "必备清单：[场景] + [数量] 件事"

8. **教程型**
   "从 0 到 1：30 天打造爆款账号"
   "从 0 到 1：[时间] + [目标]"

9. **故事型**
   "我是如何在 3 个月内赚到第一个 100 万的"
   "我是如何 + [时间] + [成就]"

10. **承诺型**
    "用这个方法，你也能日引 1000+ 精准粉"
    "用这个方法，你也能 + [结果]"
```

## 社交媒体文案

### 平台文案特点

```markdown
## 各平台文案风格

### 微信公众号
- 字数: 1500-3000 字
- 风格: 深度、专业、有料
- 结构: 开头设悬念，中间有干货，结尾有金句
- 重点: 标题决定打开率，内容决定分享率

### 小红书
- 字数: 300-800 字
- 风格: 真实、种草、攻略
- 结构: 痛点+方案+效果+总结
- 重点: 首图决定点击，关键词决定流量

### 抖音/短视频
- 字数: 50-150 字（脚本）
- 风格: 快节奏、有冲突、有反转
- 结构: 3秒抓人+内容递进+强CTA
- 重点: 前 3 秒决定完播率

### B站
- 字数: 视频为主，文案为辅
- 风格: 年轻、有梗、专业
- 结构: 开场金句+知识点+互动问题
- 重点: 一键三连引导

### 微博
- 字数: 50-280 字
- 风格: 热点、话题、互动
- 结构: 观点+论据+话题标签
- 重点: 热搜话题借势
```

### 文案模板

```markdown
## 小红书种草文案模板

【标题】
[emoji] [痛点关键词] | [解决方案] [emoji]

【正文】
开头（痛点共鸣）:
姐妹们！[痛点描述] 是不是你们也有这个困扰？

中间（解决方案）:

1️⃣ [方法1]
2️⃣ [方法2]
3️⃣ [方法3]

结尾（总结+互动）:
总之，[核心价值]！

【标签】
#[核心词] #[场景词] #[人群词] #[平台热词]
```

## SEO 文案优化

### 关键词布局

```markdown
## SEO 关键词策略

### 关键词类型
| 类型 | 特点 | 示例 |
|------|------|------|
| 核心词 | 高流量，高竞争 | 项目管理软件 |
| 长尾词 | 低流量，精准度高 | 免费项目管理软件推荐 |
| 品牌词 | 搜索量稳定 | Notion 教程 |
| 问题词 | 用户意图明确 | 如何提高团队效率 |

### 关键词布局位置
1. **标题 (Title)** - 必须包含主关键词
2. **H1 标签** - 与标题呼应
3. **首段** - 前 100 字内出现关键词
4. **H2/H3** - 自然融入长尾词
5. **正文** - 密度 1-3%
6. **图片 Alt** - 描述性关键词
7. **Meta Description** - 吸引点击

### Meta Description 模板
[核心关键词] - [价值承诺]。[补充说明]。[CTA]

示例:
"项目管理软件 - 让团队协作效率提升 200%。支持任务分配、
进度追踪、文件共享等功能。免费试用 14 天！"
```

## 邮件营销文案

### 营销邮件模板

```markdown
## 营销邮件结构

**主题行** (决定打开率)
- 简短有力 (6-10 个字)
- 包含利益点或紧迫感
- 避免垃圾邮件敏感词

**预览文本** (补充主题行)
- 承接主题行
- 增加点击动力

1. 问候语（个性化）
2. 价值开头（为什么要读）
3. 核心内容（简洁明了）
```

## 最佳实践

```
✅ 推荐
□ 先了解目标受众再动笔
□ 一个内容只传达一个核心信息
□ 使用具体数字增加可信度
□ CTA 要明确且易于执行
□ A/B 测试找出最优版本

❌ 避免
□ 自嗨式文案（只说自己好）
□ 过度承诺无法兑现的价值
□ 复杂的句式和专业术语
□ 忽视移动端阅读体验
□ 抄袭竞品文案
```

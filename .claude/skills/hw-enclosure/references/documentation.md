# Enclosure Documentation（尺寸图纸 + 装配指南）

专业 PDF 交付。工具链：D2（图）+ Typst（PDF）。

## Step 1: Dimension Drawing（D2 三视图尺寸标注）

1. 正视图：总宽×总高 + 连接器开口位置
2. 侧视图：总深×总高 + 按钮/开关位置
3. 俯视图：总宽×总深 + 安装孔位置

每个视图标注：
- 外形尺寸（±0.5mm）
- 关键特征尺寸（开口、孔位、壁厚）
- 公差标注（配合面标注 ±公差值）
- 比例尺标注

用 D2 的 near/连接线模拟尺寸标注线。产出 `dimension-drawing.svg`。

## Step 2: Cross Section（D2 截面图）

1. 纵向截面：壁厚、内腔高度、PCB 安装高度、螺丝柱结构
2. 横向截面：卡扣/密封结构细节

标注：
- 壁厚尺寸
- PCB 到顶/底间隙
- 卡扣悬臂长度和偏转量
- O-ring 槽尺寸（如有密封需求）

## Step 3: Assembly Guide（Typst 装配指南 PDF）

1. 封面：产品名称、版本、日期
2. BOM 物料清单表格
3. 工具需求清单
4. 分步装配指南（每步配图）：
   - 步骤编号 + 文字描述
   - 施力方向箭头说明
   - 注意事项（⚠️ 标注）
5. 故障排除（常见问题 + 解决方案）
6. 附录：打印参数快速参考

Typst 模板要求：
- A4 横版（装配图需要宽幅）
- 步骤编号用大字号圆形标记
- 尺寸数据用等宽字体

## Step 4: Compile Documentation（编译）

1. `typst compile assembly-guide.typ assembly-guide.pdf`
2. 验证 PDF 页数和内容完整性
3. 确认所有 SVG 图片正确嵌入
4. 检查尺寸标注值与 .scad 参数一致

如果有不一致 → 修改文档 → 重新编译。

## Step 5: Verify Consistency（文档与设计一致性验证）

1. 提取 .scad 文件中的关键参数值
2. 提取文档中的尺寸标注值
3. 逐项比对：
   - 外形尺寸一致
   - 壁厚一致
   - 安装孔位置一致
   - 连接器开口尺寸一致
   - BOM 与实际零件一致

不一致项 → 修改文档/设计 → 重新验证。产出一致性检查清单文档。

## Pass/Fail Criteria

- 三视图 + 截面图覆盖所有关键尺寸
- 装配指南每步 ≤1 个动作，配有示意图
- PDF 排版专业（A4、等宽数字、尺寸标注线）
- 文档中的尺寸与 .scad 参数 100% 一致
- BOM 物料清单完整（含规格和来源）
- 编造尺寸 = FAIL（必须与 .scad 源文件交叉验证）

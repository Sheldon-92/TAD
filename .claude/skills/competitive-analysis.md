# Competitive Analysis Skill

> 综合自战略分析和市场研究最佳实践，已适配 TAD 框架

## 触发条件

当用户需要分析竞争对手、评估市场格局、制定竞争策略或进行行业研究时，自动应用此 Skill。

## 最佳实践

```
✅ 推荐
□ 定期更新竞品信息，保持时效性
□ 结合多个框架全面分析
□ 用数据和事实支撑结论
□ 分析要转化为可执行建议
□ 区分事实和推测

❌ 避免
□ 只看功能不看战略
□ 过度关注竞品忽视用户
□ 分析后不采取行动
□ 使用过时的信息
□ 主观臆断无依据
```

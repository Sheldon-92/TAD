# Skill Library (T2 — Reference Shelf)

Harvested working patterns from downstream projects, promoted from T1 (project-local
SCAND candidates) after the in-session human confirmation ceremony.

## Tier Model

| Tier | Location | Scope | Promotion Rule |
|------|----------|-------|----------------|
| T1 | `.tad/active/skillify-candidates/` | Project-local draft | Blake writes SCAND after 4-gate KA |
| T2 | `.tad/skill-library/` (here) | Cross-project reference | Human confirms in-session (Phase 2 ceremony) |
| T3 | `.claude/skills/` | Production skill | Requires ≥2-project evidence (Domain Pack decision rule) |

## Rules

- **NEVER distributed** — this directory is zero-touch deny-listed in both
  `derive-sync-set.sh` and `tad.sh`. It does not sync to downstream projects.
- **Promotion to T3 (packs/skills)** requires ≥2-project evidence per the
  Domain Pack vs Project-Knowledge decision rule.
- Entries are indexed in `_index.md`.

## Pattern
Batch generation → per-item evaluation page → precise iteration (human-AI evaluation paradigm). When users need to evaluate, annotate, and filter a batch of audio/materials/generated results one by one: generate an interactive HTML page with per-item cards, verdict buttons, and export JSON.

## Note
Superseded-in-spirit by the TAD Feedback Collector (v2.28.0), which generalized this pattern into a universal structured feedback protocol with overlay and card modes. Kept as origin reference documenting where the concept was first discovered.

## T3 Status
Not a T3 candidate — the generalized version already exists as a TAD-level protocol (feedback_collector_protocol in blake/SKILL.md).

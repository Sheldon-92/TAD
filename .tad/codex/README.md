## Shared Protocol Boundary

The following are **shared TAD protocol** (in SKILL body, not in Codex config):

- Alex/Blake role contracts, Gates 1-4, handoff discipline
- Layer 2 review criteria and group ordering
- Ralph Loop, circuit breaker, escalation
- Completion report format, evidence requirements
- Knowledge assessment (triple-question KA)
- Execution-discipline rules (MUST/MANDATORY/VIOLATION)

Codex adapter handles **platform mechanics only**: how hooks fire, how skills load, how subagents spawn, how sandbox works. It does not define quality-chain rules.

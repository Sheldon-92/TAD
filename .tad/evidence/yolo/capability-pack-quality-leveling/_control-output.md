# CONTROL Answer (generalist, NO pack) — Full Episode Production

Scenario: 4000-word article + reference voice clip → finished podcast for Spotify and Apple, professional master.

## 1. Write the script
Read the 4000-word article and adapt it into a conversational podcast script. Add an intro hook,
clear segments, and an outro. Keep it natural and engaging for listeners. Proofread for any errors
and tighten the language so it reads well aloud.

## 2. Generate the narration
Use VoxCPM2 with the reference voice clip to clone the voice. Feed the script in and generate the
narration audio. Break the script into reasonable segments so the model handles it well, then
generate each part. Clean up any background noise and make sure the audio is clear.

## 3. Add background music
Pick a suitable royalty-free background track that matches the tone. Lower the music volume under
the narration so the voice stays clear (duck the music when the host is talking), and bring it up
between segments. Add fade-ins and fade-outs at the start and end.

## 4. Master for Spotify and Apple Podcasts
Normalize the loudness to a standard podcast level and make sure nothing clips. Export a clean,
balanced mix so it sounds professional and consistent across both platforms. Do a final listen-through
to confirm the levels and quality before publishing.

Usage: notebooklm source stale [OPTIONS] SOURCE_ID

  Check if a URL/Drive source needs refresh.

  Returns exit code 0 if stale (needs refresh), 1 if fresh. This enables shell
  scripting: if notebooklm source stale ID; then refresh; fi

  SOURCE_ID can be a full UUID or a partial prefix (e.g., 'abc' matches
  'abc123...').

  Examples:
    source stale abc123              # Check if stale

Options:
  -n, --notebook TEXT  Notebook ID (uses current if not set)
  --help               Show this message and exit.

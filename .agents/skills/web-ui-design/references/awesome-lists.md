# Awesome Lists & Resources

Curated GitHub resources for web UI design and design systems.

---

## Design Systems

- **Awesome Design Systems** — https://github.com/alexpate/awesome-design-systems
  Comprehensive list of design systems from companies worldwide.

- **Design Systems Database** — https://designsystemsdb.com
  Searchable database of public design systems with technology and license info.

---

## Component Libraries

- **Awesome React Components** — https://github.com/brillout/awesome-react-components
  Curated list of React UI components and libraries.

- **Headless UI Libraries** — Search GitHub for `headless-ui` topic
  - Ark UI: https://github.com/chakra-ui/ark
  - Radix UI: https://github.com/radix-ui/primitives
  - Headless UI: https://github.com/tailwindlabs/headlessui

## Design Tokens

- **Design Tokens Community Group (DTCG)** — https://www.w3.org/community/design-tokens/
  W3C community defining the standard JSON format for design tokens.

- **Awesome Design Tokens** — Search GitHub for `design-tokens` topic
  Key tools:
  - Style Dictionary: https://github.com/amzn/style-dictionary
  - Tokens Studio: https://github.com/tokens-studio
  - Design Tokens Validator: validates against DTCG spec

## Accessibility

- **Awesome A11y** — https://github.com/brunopulis/awesome-a11y

- **WAI-ARIA Authoring Practices** — https://www.w3.org/WAI/ARIA/apg/

## Tailwind CSS Ecosystem

## WCAG & APCA

- **WCAG 2.1 Quick Reference** — https://www.w3.org/WAI/WCAG21/quickref/
  Filter by success criteria level (A, AA, AAA).

- **APCA Contrast Checker** — https://www.myndex.com/APCA/
  Advanced Perceptual Contrast Algorithm — replaces the 4.5:1 ratio.
  LC ≥60 for body, ≥45 for large text.

- **Colour Contrast Checker** — https://colourcontrast.cc
  Simple WCAG + APCA checker.

## Books and References

- *Refactoring UI* — Adam Wathan & Steve Schoger
  Practical design advice for developers. Covers spacing, color, typography.

- *Design Systems* — Alla Kholmatova
  Deep dive into building and maintaining design systems at scale.

- *Inclusive Design Patterns* — Heydon Pickering

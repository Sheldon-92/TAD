# Ergonomics & Appearance（人机工程 + 外观 + 手感）

基于人体测量学数据，不是凭经验。研究型能力：search → analyze → derive → generate。

## Step 1: Search Ergonomics（人机工程数据）

1. 握持方式：单手/双手/桌面放置/壁挂/佩戴
2. 手掌尺寸数据（ISO 7250）：
   - 成人手掌宽度 P50: 男 83mm / 女 74mm
   - 握力 P5: 男 275N / 女 166N
   - 拇指操作范围：单手持握时 ≤70mm
3. 搜索同类产品的尺寸和人体工学评测
4. 搜索目标用户群体的特殊需求（老人→大按钮、防滑；儿童→圆角、无小零件）

参考搜索词：
- `"handheld device ergonomics" dimensions grip comfort`
- `"IoT device form factor" user study`
- `"{产品类型}" ergonomic design guidelines`
- `"enclosure surface finish" texture grip comparison`

## Step 2: Analyze Form Factor（外形因素分析）

1. 尺寸约束：
   - 最大外形 = 人体工学限制 vs 内部元器件需求
   - 如果内部元器件大于人体工学最佳尺寸 → 必须权衡
2. 握持舒适度：
   - 边缘圆角 ≥3mm（长时间握持不割手）
   - 背面曲率匹配手掌弧度（R=80-120mm）
   - 重心位置（偏下 = 稳定感，偏上 = 翻倒风险）
3. 操作界面布局：
   - 按钮位置在拇指自然触达范围内
   - 显示屏可视角度 ≥30°（倾斜使用时）
   - LED 指示灯不在强光直射面
4. 表面处理：
   - 哑光 = 防指纹 + 防滑
   - 亮面 = 高级感但滑
   - 纹理 = 最佳手感但增加模具成本

质量要求：人体尺寸数据必须引用标准（ISO 7250）或搜索来源，不是凭经验。

## Step 3: Derive Design Language（外观设计语言）

1. 设计风格关键词：3-5 个形容词（如：极简/坚固/科技感/自然/可爱）
2. 色彩方案：
   - 主色 + 辅色 + 强调色（Hex 值）
   - 色彩心理学依据（白=清洁, 黑=专业, 绿=环保）
3. 分型线处理：
   - 隐藏在边缘圆角处（不在正面可见位置）
   - 与表面纹理方向一致（减少视觉突兀）
4. 标识/Logo 位置和方式：
   - 凸刻（FDM 友好）vs 凹刻（注塑友好）vs 丝印
5. 参考产品图片（搜索 3+ 个同类优秀设计）

## Step 4: Generate Report

综合为人机工程报告 PDF（含尺寸分析、外观定义、参考图）。

## Pass/Fail Criteria

- 人体尺寸数据引用 ISO 7250 或等效标准
- 边缘圆角 ≥3mm（长时间握持）
- 按钮在拇指自然触达范围内（单手 ≤70mm）
- 设计语言有 3-5 个明确关键词 + 色彩方案
- ≥3 个参考产品设计（有图片来源）
- 编造人体数据 = FAIL

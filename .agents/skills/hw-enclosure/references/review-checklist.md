# Expert Review Checklists（专家审查清单）

每个能力交付后，用对应 persona 的清单做审查（review 时才用 persona，执行时不用）。

## 机械工程师 — Enclosure Design 交付审查

- 壁厚是否满足制造方式的最小值？
- 配合面公差是否正确？
- 螺丝柱结构是否合理（外径、高度、加强筋）？
- 悬空角度是否在 45° 以内？
- .scad 文件能否无错误编译导出 STL？

## 硬件工程师 — PCB Fitting 交付审查

- 安装柱尺寸是否匹配螺丝规格？
- 连接器开口位置是否与 PCB 实际位置对齐？
- 电池和 PCB 之间是否有足够间隙？
- 天线区域是否避开了金属结构？

## 材料工程师 — Material Selection 交付审查

- 材料参数是否来自正规数据手册？
- 决策矩阵权重是否合理？
- 成本估算是否包含模具/后处理？
- 阶段建议是否考虑了转换成本？

## 生产工程师 — Assembly Design 交付审查

- 装配顺序是否合理（先内后外、先重后轻）？
- 紧固方式是否匹配材料和壁厚？
- 是否考虑了批量生产的工时优化？
- 密封设计是否满足 IP 等级要求？

## 工业设计师 — Ergonomics 交付审查

- 外形尺寸是否符合人体工学？
- 操作界面布局是否方便单手使用？
- 设计语言是否与品牌一致？
- 分型线是否隐藏在不显眼位置？

## 3D 打印工程师 — Manufacturing Export 交付审查

- STL 文件是否无非流形错误？
- 打印方向是否最优（最少支撑+最高强度）？
- 打印参数是否匹配材料和零件特征？
- 文件包是否完整（STL+源文件+参数+BOM）？

## 技术文档工程师 — Documentation 交付审查

- 三视图是否完整（正/侧/俯）？
- 截面图是否标注了壁厚和内部结构？
- 装配指南是否清晰可执行？
- 所有尺寸是否与 CAD 源文件一致？

## 端到端验收样例（源自原 pack 的 gate 测试题）

测试题：Wayo 大象追踪器外壳 — ESP32-C3 + 5.65 inch E-ink + 18650 电池。
约束：ESP32-C3 DevKit (18×20mm)、E-ink 模组 122×95mm（活动区 114.9×85.8mm）、18650 电池侧放 (65×18mm)、USB-C + SMA 天线座 + 电源开关、户外 -10~55°C 防雨 IP54+、原型 FDM (PETG) / 小批量 SLA、E-ink 窗口透光保护、天线净空、电池可更换。

分能力验收点：
- enclosure_design: .scad 无错误编译（`openscad -o test.stl --render`）；壁厚 ≥1.5mm (PETG FDM)；E-ink 开口 ≥ 活动区域；所有尺寸可 `-D` 覆盖
- pcb_fitting: 安装柱匹配数据手册；FPC 有 90° 弯折空间；USB-C 开口 9.5×3.5mm 含公差；SMA 开口 + 天线净空无金属
- material_selection: PETG 选型有工程参数支撑（耐温 ≥70°C、耐紫外优于 PLA）；成本含材料+打印时间；有户外耐候数据（ASA 备选）
- assembly_design: 电池可更换（无需工具或仅一字螺丝刀）；装配 ≤5 步；有爆炸图
- manufacturing_export: 底壳/顶盖/电池盖分别导出 STL；STL 尺寸与设计一致；打印参数匹配 PETG + 零件特征
- documentation: 三视图覆盖所有外形尺寸；装配指南含 BOM + 分步说明；文档尺寸与 .scad 一致

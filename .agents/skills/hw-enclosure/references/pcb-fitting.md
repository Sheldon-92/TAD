# PCB Fitting（PCB 安装定位）

安装柱 + 连接器开口 + 线缆走线空间。

## Step 1: Measure PCB（精确测量，±0.1mm 精度）

1. PCB 外形尺寸：长×宽×厚（含 FR4 + 焊盘高度）
2. 安装孔：数量、直径、中心坐标（以 PCB 左下角为原点）
3. 最高元器件高度（正面/背面分别）
4. 连接器列表：

   | 名称 | 类型 | 中心 X | 中心 Y | Z(从PCB底面) | 开口宽 | 开口高 | 朝向 |

5. 天线位置和净空要求（金属外壳需要天线窗口）
6. 散热元件位置（需要通风孔或散热柱对齐）

数据来源优先级：数据手册 > KiCad/EDA 文件 > 实测。产出记录到 `pcb-fitting.md` 类分析文档。

## Step 2: Generate Mounting（安装结构）

在 .scad 中添加 PCB 安装结构：

1. 安装柱（standoff）：
   - 高度 = PCB 底面净空需求（通常 3-5mm）
   - 外径 = 安装孔直径 × 2.5（M2 孔 → OD=5mm, M3 孔 → OD=7.5mm）
   - 内孔 = 自攻螺丝预钻孔径（M2=1.6mm, M3=2.5mm FDM）
   - 顶部加 0.5mm 倒角（引导 PCB 放入）
2. 定位销（alignment pin）— 无螺丝方案：
   - 2 个定位销 + 2 个安装柱（对角布局）
   - 销直径 = 安装孔直径 - 0.3mm（过盈配合）
3. PCB 边缘导槽（可选）：
   - 槽宽 = PCB 厚度 + 0.3mm
   - 槽深 ≥ 1.5mm

⚠️ 安装柱位置必须与 Step 1 的坐标精确对应。

## Step 3: Connector Cutouts（连接器开口）

在外壳侧面/底面开连接器窗口：

1. 每个连接器开口：
   - 位置 = 连接器中心坐标 + PCB 安装高度 + 壳壁偏移
   - 宽度 = 连接器宽 + 1.0mm（FDM）或 + 0.5mm（SLA）
   - 高度 = 连接器高 + 0.8mm
   - 圆角 = 0.5mm（减少应力集中）
2. USB-C 开口标准尺寸：9.5mm × 3.5mm（含公差）
3. LED 导光孔：直径 = LED 直径 + 0.5mm，深度 = 壁厚 - 0.5mm（留薄层导光）
4. 按钮孔：直径 = 按钮直径 + 0.3mm，加防尘唇 0.3mm
5. E-ink/LCD 显示窗：开口 = 可视区域 + 1mm 每边，加 0.8mm 台阶压屏

⚠️ 开口位置误差超过 ±0.5mm = 连接器无法插入或外观丑陋。

## Step 4: Verify Fitting（装配可行性验证）

1. 用 OpenSCAD 的 `#` / 透明修饰符检查 PCB 与壳体的空间关系。在 .scad 中添加 PCB 占位块（透明）：
   ```
   #color("green", 0.3) translate([pcb_x, pcb_y, pcb_z]) cube([pcb_l, pcb_w, pcb_h]);
   ```
2. 检查清单：
   - PCB 四边间隙 ≥1mm
   - 最高元器件到顶盖间隙 ≥1.5mm
   - 连接器开口与连接器对齐（渲染侧视图确认）
   - 线缆弯曲半径 ≥ 线缆直径×3
   - 天线区域无金属遮挡
3. 渲染截面图确认内部空间

如果间隙不足 → 调整参数 → 重新渲染。

## Step 5: Optimize Cable Routing（线缆走线空间）

1. 电池连接线：预留弯曲半径 ≥ 线径×3 的通道
2. FPC 排线（如屏幕）：预留 90° 弯折空间 + 最小弯曲半径
3. 天线馈线：避开大电流走线，预留 SMA/U.FL 连接器操作空间
4. 如果空间紧张：在壳体内壁加走线槽（宽=线径+1mm, 深=线径）
5. 装配顺序标注：哪根线先接，避免"装不进去"

## Pass/Fail Criteria

- 安装柱位置与 PCB 安装孔坐标误差 ≤0.1mm
- 连接器开口尺寸包含制造公差（FDM +1.0mm, SLA +0.5mm）
- PCB 四边间隙 ≥1mm，顶部间隙 ≥1.5mm
- 有 PCB 占位块透明渲染验证
- 线缆弯曲半径 ≥ 线径×3
- CNC 加工内角使用 Dogbone 减压槽，FDM 水平孔使用 Teardrop 形状 — 来源: NopSCADlib DFM modules
- 编造 PCB 尺寸 = FAIL。必须引用数据手册或实测数据
